# 開発タスク計画
2025/06/18 07:30

## プロジェクト概要
かうめも（kau-memo）は、どうぶつの森風のゲーミフィケーション要素を持つ家族・グループ向け買い物リスト管理アプリです。

**優先度: 最高**
